# Decode Labs Data Science Internship

This repository contains the four weekly projects I completed during my Data
Science Internship at Decode Labs in May and June 2026.

Each week focused on a different part of the data science workflow, from
exploratory analysis and classical machine learning to customer intelligence
and federated learning for medical imaging.

## Weekly Projects

### Week 1 - Heart Disease Analysis

This project analyzes the UCI Cleveland heart disease dataset with exploratory
data analysis and predictive modeling.

Main themes:
- Data cleaning and preparation
- Exploratory data analysis
- Clinical feature comparison
- Classification modeling
- Model evaluation and feature importance

Key ideas practiced:
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Logistic Regression
- Decision Tree
- Random Forest
- ROC-AUC and balanced accuracy

### Week 2 - Hospital Readmission Prediction

This project predicts 30-day hospital readmission in diabetic patients using the
UCI Diabetes 130-US Hospitals dataset.

Main themes:
- Data cleaning
- Feature engineering
- Patient-aware train/test splitting
- Imbalanced classification
- Model comparison
- Fairness and subgroup analysis

Key ideas practiced:
- Data preprocessing
- Logistic Regression
- Random Forest
- Gradient Boosting
- XGBoost
- Recall, precision, F1-score, ROC-AUC
- Bootstrap confidence intervals

### Week 3 - E-Commerce Customer Intelligence

This project analyzes the Olist e-commerce dataset to study customer behavior,
delivery performance, and review score prediction.

Main themes:
- Customer segmentation with RFM analysis
- Delivery delay analysis
- Review score prediction
- Tabular machine learning
- Interpretability and error analysis

Key ideas practiced:
- Data aggregation and feature engineering
- RFM segmentation
- Logistic Regression
- Random Forest
- XGBoost
- Accuracy, macro-F1, and confidence intervals
- Plotting and reporting results

### Week 4 - Federated Learning for Tuberculosis Screening

This project studies privacy-preserving federated learning for tuberculosis
detection from chest X-rays.

Main themes:
- Medical image classification
- Federated learning with Flower
- Differential privacy with Opacus
- Centralised versus federated comparison
- Error analysis and interpretability

Key ideas practiced:
- ResNet-18
- Federated averaging
- Differential privacy
- Bootstrap evaluation
- McNemar test
- GradCAM
